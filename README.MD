# cat-tools

## 简介

根绝公司项目需求，将常用方法抽离出来，便于后续的方法的统一拓展。

详情参考[线上文档](https://demo.assetech.com.cn/index.html)

## 开发

```javascript
# 克隆项目
git clone https://github.com/CrazyStudent13/cat-tools.git

# 进入项目目录
cd cat-tools

# 安装依赖
npm install
```

## 发布

```javascript
# 生成文档
npm run doc

# 打包
npm run build

```
